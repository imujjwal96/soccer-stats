Pro Evolution Soccer 2018 - Player Statistics

Context

Dataset for people who love have grown up playing PES.

Content

  • Every player featuring in PES 2018
  • 70+ attributes
  • Player and Flag Images
  • Playing Position Data
  • Attributes based on actual data of the latest Konami's PES 2018 game
  • Attributes include on all player style statistics like Dribbling, Aggression, GK Skills etc.
  • Player personal data like Nationality, Photo, Club, Age, Wage, Salary etc.

Data Source

The data is scraped from the website https://pesmaster.com by extracting the Player personal data and Player Ids and then the playing and style statistics.

Acknowledgements

The data has been crawled from the https://pesmaster.com website.
